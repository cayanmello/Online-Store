## 💻 Projeto

Aplicação desenvolvida no curso da [Trybe](https://www.betrybe.com/). Neste projeto foi criado uma versão simplificada, sem persistência no banco de dados, de uma **loja online**, desenvolvendo suas funcionalidades de acordo com demandas definidas em um quadro _Kanban_. É possível buscar produtos, adicionar ao carrinho de compras e visualizar detalhes utilizando a _API do Mercado Livre.

## 🚀 Tecnologias
Esse projeto foi desenvolvido com as seguintes tecnologias:

Front-end:
> Desenvolvido usando: React, Redux, CSS3, HTML5, ES6
